# Nora Angel

Core home for Nora's durable state and operating scaffolding.

## Purpose

This repo is intentionally small and human-restorable.

It is for:
- documentation
- config scaffolding
- operational notes
- lightweight scripts
- personality / identity artifacts
- restore and recovery notes

It is **not** for:
- large datasets
- bulky exports
- disposable caches
- huge binaries that take forever to sync

## Backup philosophy

- **GitHub** keeps the important lightweight artifacts versioned.
- **Mega** holds emergency Hermes backups for restore.
- **Large data stays elsewhere** unless it is truly worth preserving.
